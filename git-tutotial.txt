initially push the code in to github
---------------------------------------------------------------
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/Ratnadeepdirsipam/practise.git
git push -u origin main

---------------------------------------------------
push an existing repository from the command line
--------------------------------------------------------------
git add path-to-file
git remote add origin https://github.com/Ratnadeepdirsipam/practise.git
git branch -M main
git push -u origin main
